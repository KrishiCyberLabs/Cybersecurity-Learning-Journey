# Defensive Security Intro

## Overview

This room provided an introduction to the fundamentals of defensive security and the strategies organizations use to protect their digital assets. It explored how security teams detect threats, monitor network activity, and respond to incidents to maintain a secure environment. The hands-on examples helped demonstrate the critical role defensive security plays in modern cybersecurity operations.

## Topics Explored

* Defensive Security Fundamentals
* Security Monitoring
* Threat Detection
* Incident Response
* Security Operations Center (SOC)
* Log Analysis

## What I Learned

A key lesson from this room was that cybersecurity is not only about preventing attacks but also about identifying and responding to threats as quickly as possible. I learned how security analysts continuously monitor systems for suspicious activity and use logs to investigate potential security incidents.

The room also introduced the importance of Security Operations Centers (SOCs), where security professionals work together to detect, analyze, and respond to cyber threats. Understanding how alerts, logs, and monitoring systems work provided valuable insight into the daily responsibilities of defensive security teams.

* Security Operations Understanding
* Analytical Investigation Skills

## Tools and Concepts

* Security Information and Event Management (SIEM)

* Incident Response Processes

## Personal Reflection

This room gave me a deeper understanding of how organizations defend against cyber threats on a daily basis.
